This directory stores result tables and archived outputs used by the thesis.

Current contents:
- final_resample_raw/: full raw per-fold outputs for the 52-model final resampling study, plus combined prediction and fold-metric tables
- final_resample_subgroups/: derived subgroup summaries used for the interaction-class analysis in the report
- archive_prodigy/: saved PRODIGY estimate files from archived branches
- archive_ca_contact/: saved CA contact-map regression summaries and prediction files
- archive_mmgbsa/: saved MM/GBSA shard outputs, failure logs, and merged shard table

The main machine-readable summary for the final 52-model comparison is in:
- final_resample_matrix_summary.csv
