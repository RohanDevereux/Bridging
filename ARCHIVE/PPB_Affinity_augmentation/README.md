# PPB-Affinity Augmentation

Archived PPB-inspired spatiotemporal branch.

Contents:

- `src/bridging/ppb`
  - Frame, trajectory, and ensemble training code inspired by PPB-Affinity-style setups
- `src/BaselineModel`
  - Vendored external model stack used by that branch
- `hpc/`
  - HPC training launcher

This branch explored direct PPB-style learning on MD-derived examples. It is preserved for appendix material, but it is not part of the final MD -> GraphVAE pipeline.
