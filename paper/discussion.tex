\chapter{Discussion and Conclusions}
\label{ch:discussion}

On the 1437-complex evaluation set, the strongest model was the direct supervised baseline on the full static residue graph. The full view outperformed the interface-restricted view consistently, the molecular-dynamics-derived summaries did not improve held-out performance over the static feature set, and the direct supervised baseline outperformed the best GraphVAE configuration.

The graph-view comparison is unambiguous. The full graph won all 24 matched GraphVAE comparisons and both supervised baseline comparisons. In this implementation, the interface-restricted view removed useful context.

The dynamic comparison is specific to the implemented feature set: hydration summaries, C$\alpha$ RMSF, torsion summaries, contact frequencies, distance summaries, and approximate inter-chain force proxies attached to a static residue graph. Under that encoding, the dynamic channels did not produce a held-out gain.

The GraphVAE learned usable latent representations, but the best predictive model was still the supervised baseline. On this dataset and with this encoder backbone, masked reconstruction and latent regularization did not yield the strongest held-out predictor.

The main limitations are the short trajectory length, the static graph topology, the use of time-aggregated dynamic features rather than an explicitly temporal model, and evaluation on one dataset family. A practical limitation is dataset scale: the processed broad table contained 2818 curated complexes, but the full MD and repeated-resampling study was run on a smaller compute-bounded analysis subset. PPB-Affinity reported better benchmark performance as the training fraction increased, so this restricted study size matters \citep{Liu2024PPBAffinity}. A second limitation is the split itself. The resampling was repeated, but it was not family-disjoint. Earlier protein--protein work explicitly removed homologous complexes to avoid optimistic evaluation, and protein--ligand studies have shown that structural overlap between training and test sets can inflate reported affinity-prediction performance \citep{Moal2011AffinityPrediction,Graber2025DataBias}. Natural extensions are therefore larger MD-backed study sets, longer simulations, models that operate directly on temporal graph sequences, and stricter split definitions.

Subtype heterogeneity also remained visible in the final sweep. On pooled outer-test predictions, the best overall model (\texttt{full\_base\_S}) gave Pearson $r=0.292$ on antibody--antigen, $r=0.430$ on the broad other class, and $r=0.466$ on TCR-pMHC. The full static baseline remained the strongest configuration in all three interaction classes, and the interface-restricted models remained weaker in all three.

Within the implemented pipeline, short explicit-solvent molecular-dynamics summaries did not improve protein--protein binding affinity prediction over a static residue graph, and the strongest overall model was the direct supervised baseline on the full static graph.
