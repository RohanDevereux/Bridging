\chapter{Repository Layout}
\label{ch:code}

The source code for this thesis is hosted at \url{https://github.com/RohanDevereux/Bridging}. The active molecular-dynamics preparation and simulation code is under \path{src/bridging/MD}. The residue-graph preparation code is under \path{src/bridging/graphvae/prep}. The GraphVAE, supervised baseline, and latent-ridge evaluation code are under \path{src/bridging/graphvae/ml}. The command-line runners used for the final evaluation are under \path{src/bridging/graphvae/runners}. Shared affinity-conversion and dataset utilities are under \path{src/bridging/utils}.

The curated datasets used in the main pipeline are under \path{src/bridging/processedData}. The compact summary of the final 52-configuration matrix is \path{RESULTS/final_resample_matrix_summary.csv}. The saved subgroup analysis derived from the raw outer-fold prediction files is under \path{RESULTS/final_resample_subgroups}. The script that rebuilds those subgroup summaries from extracted raw predictions is \path{src/bridging/graphvae/tools/export_final_resample_subgroups.py}. Archived branches discussed in Appendix~\ref{ch:appendix_archive} are under \path{ARCHIVE}.
